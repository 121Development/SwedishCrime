# SwedishCrime

Program will be modular
1st module will be a RSS scraper
2nd module store do DB
3rd publish to website


1st module will be a RSS scraper
    Chose RSS scrape tool
    Adapt that to specific RSS feed
        Time-module, search each minute or every two minutes
    read DB, if new, store in DB
    Geoparse location names and map to coordinates or places on google map or geoparse all names

2nd module store do DB
    Set keywords to store
    Filter non keywords
    Store all data
    Store only new data


3rd publish to website
    Auto detect regions/cities
    Website filter options
    Subscribe for updates from your region of choice
    Map templates

4th Monetize website
    Email + SMS alerts
    Ads
    Protection PDFs
